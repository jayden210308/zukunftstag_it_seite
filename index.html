START-TEAM

Du siehst immer nur 1 Rätsel. Wenn es stimmt, kommt automatisch das nächste.

Aktuelles Rätsel

Rätsel 1: Teamname

Öffne Untersuchen und suche im HTML nach AUFGABE 1.
Ändere die Überschrift so, dass sie ein passender Teamname ist.
Tipp: 2 Wörter, mit „-“ dazwischen. Nicht raten: Überleg, worum es heute geht.

Rätsel 2: Himmel

Suche im CSS nach AUFGABE 2.
Setze den Hintergrund auf die Himmelsfarbe, aber als HEX.
Tipp: HEX sieht so aus: #12ab34. Die Himmelsfarbe heißt oft „skyblue“.

Rätsel 3: Sonne

Suche im CSS nach AUFGABE 3.
Die Button-Farbe soll wie die Sonne aussehen – aber wieder als HEX.
Tipp: „gold“ als HEX finden (oder merken): es beginnt mit #FF…

Rätsel 4: Geheimwort

Suche im HTML nach AUFGABE 4.
Trage als Geheimwort das erste Wort aus der Überschrift ein (vor dem „-“), aber klein geschrieben.
Beispiel: Überschrift „HELLO-WORLD“ → Geheimwort wäre „hello“.

Rätsel 5: Zahlencode

Suche im HTML nach AUFGABE 5.
Rechne: Anzahl Buchstaben vom Geheimwort × 7
Trage das Ergebnis als Zahl ein (nur Ziffern).
Tipp: Nicht raten – zählen!

Rätsel 6: Freischalten

Suche im HTML nach AUFGABE 6.
Setze das Freischalt-Wort auf: FREI (genau so geschrieben).
Tipp: Danach bist du fertig.
Fortschritt wird automatisch geprüft…
