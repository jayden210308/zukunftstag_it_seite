👨‍💻 Code Mission

Lerne Schritt für Schritt, wie Funktionen aufgebaut sind

Aufgabe 1: Begrüßung
Deine Funktion soll genau diesen Text zurückgeben:
👉 Hallo Welt

So ist eine Funktion aufgebaut:
function name(){ return "TEXT"; }
💡 Du musst:
  • return benutzen
  • den Text in "Anführungszeichen" schreiben
Aufgabe 2: Rechnen
Deine Funktion bekommt zwei Zahlen: a und b
👉 Du sollst sie multiplizieren
function name(a,b){ return a * b; }
Aufgabe 3: Text verändern
Du bekommst einen Text (z.B. "test")
👉 Mach ihn GROSS
text.toUpperCase()

🧠 Grundlagen

Funktion
Ein Codeblock, der etwas macht.
return
Gibt ein Ergebnis zurück.
Strings
Text steht in "...".
*
Das ist das Mal-Zeichen.
