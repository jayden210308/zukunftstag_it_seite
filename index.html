Aufgabe 1: Name eingeben

Schreib deinen Namen rein und klick auf „Weiter“.


Begriff: HTML ist der Inhalt (Texte, Buttons, Bilder).

Aufgabe 2: Button klicken

Klick auf den Button. Danach darfst du weiter.


Begriff: JavaScript ist „Aktion“ (was passiert beim Klicken).

Aufgabe 3: Dropdown auswählen

Wähle ein Gerät aus.

Idee: Später kannst du im HTML eine neue Option hinzufügen (z.B. Smartwatch).

Aufgabe 4: Bild an/aus

Klick auf den Button, um das Bild zu verstecken/zu zeigen.


Idee: Später kannst du im HTML beim Bild src="..." ändern.

Aufgabe 5: „Untersuchen“ (HTML)

Rechtsklick auf die Seite → Untersuchen.
Suche im HTML oben die <h1> Überschrift und ändere sie (z.B. dein Name).

✅ Super! Du hast etwas im HTML geändert.

Wichtig: Das ist nur bei dir sichtbar. Nach dem Neuladen ist es wieder wie vorher.

Aufgabe 6: „Untersuchen“ (CSS)

Öffne wieder Untersuchen.
Suche im CSS :root und ändere eine Farbe, z.B. --accent.

🎉 Geschafft!
Bonus-Ideen:
  • Ändere den Button-Text („Bild an/aus“) im HTML.
  • Füge im Dropdown „Smartwatch“ hinzu.
  • Tausche das Bild (anderer Link oder eigenes Bild).
Begriff: CSS ist das Aussehen (Farben, Schrift, Größen).
