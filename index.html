Missionen (klar & getrennt)

Aufgaben

Aufgabe 1: Überschrift ändern (HTML)

  • Suche <h1> oben und schreibe deinen Namen rein.
  • Idee: „Mias Hacker-Seite“, „Tim baut Webseiten“ …

Aufgabe 2: Farben ändern (CSS)

  • Suche im CSS :root.
  • Ändere --bg, --main oder --accent.

Aufgabe 3: Text per Button ändern (JavaScript)

  • Klicke „Text ändern“.
  • Suche im HTML id="dynamicText" und ändere den Text.
  • Idee: Mach einen Witz oder deinen Lieblingsspruch.

Aufgabe 4: Bild tauschen (HTML)

  • Suche beim Bild src="..." und setze einen anderen Link ein.
  • Idee: Tier / Fußball / Auto / Weltraum.

Aufgabe 5: Dropdown erweitern (HTML)

  • Füge eine neue Zeile hinzu, z.B.: <option value="Smartwatch">Smartwatch</option>
  • Danach auswählen und schauen was passiert.

Bonus: Eigene Idee (sehr cool)

  • Kopiere eine Box (ein <div class="card">) und baue eine neue.
  • Ideen: „Lieblingsspiel“-Dropdown, „Lieblingsessen“-Buttons, „Meine Farben“-Box.
Wenn etwas kaputt ist: Reset klicken oder Seite neu laden.
