👨‍💻 Code Mission

Finde heraus, wie Funktionen funktionieren und löse die Aufgaben

Aufgabe 1: Begrüßung
Deine Funktion soll einen Text zurückgeben.

💡 Überleg mal:
  • Wie gibt man in JavaScript etwas zurück?
  • Wie schreibt man Text (Strings)?
Aufgabe 2: Rechnen
Deine Funktion bekommt zwei Werte.

💡 Überleg mal:
  • Wie greift man auf a und b zu?
  • Welches Zeichen benutzt man zum Multiplizieren?
Aufgabe 3: Text verändern
Du sollst Text verändern.

💡 Überleg mal:
  • Gibt es Funktionen für Strings?
  • Was könnte "upper" bedeuten?

🧠 Hilfe

Funktion
Eine Funktion ist ein Block Code, der etwas macht und oft etwas zurückgibt.
return
Damit schickst du ein Ergebnis zurück.
Strings
Text wird in "Anführungszeichen" geschrieben.
Parameter
Werte wie (a, b), die in die Funktion reingegeben werden.
