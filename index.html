✅ Aufgaben (du kannst sie in beliebiger Reihenfolge machen)

Wenn etwas kaputt ist: Klick auf Reset oder lade die Seite neu.
  • Aufgabe 1: Ändere oben die Überschrift (<h1>).
  • Aufgabe 2: Ändere im CSS bei :root eine Farbe (z.B. --accent).
  • Aufgabe 3: Ändere Button-Texte (z.B. mach „Text ändern“ lustig).
  • Aufgabe 4: Füge eine neue Dropdown-Option hinzu (z.B. Smartwatch, Konsole, VR-Brille).
  • Aufgabe 5: Füge eine neue Checkbox hinzu (z.B. GPU, Mainboard, Lüfter).
  • Aufgabe 6: Ändere das Bild (anderer Link oder dein eigenes Bild).
  • Bonus: Erfinde eine neue Box mit Überschrift + Button + Text.
